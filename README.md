# 🎯 Whack-a-Mole Game

A fun and interactive **Whack-a-Mole browser game** built using HTML, CSS, and JavaScript.  
The objective is simple: hit the moles as they randomly appear and score as many points as possible before time runs out.

---

## 🚀 Features

- 🎮 Interactive gameplay  
- 🎯 Random mole appearance  
- ⏱️ Timer-based game  
- 🧮 Score tracking system  
- ⚡ Fast and responsive UI  
- 🎨 Simple and clean design  

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling  
- **JavaScript (Vanilla)** – Game logic  

---

## 🎯 How to Play

1. Start the game  
2. Moles will appear randomly  
3. Click (whack) the mole as fast as possible  
4. Earn points for each successful hit  
5. Game ends when time runs out
